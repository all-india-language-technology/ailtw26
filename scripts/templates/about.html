About This Course

What is this?

This site contains the course materials for Prompt Engineering for Bible Translators, prepared for the AILT 2026 AI Tools Track workshop.

The course is built around the LtL (Learning to Learn) framework — a learner-centred adult education approach emphasising dialogue, problem solving, and attitude/skill/knowledge (ASK) transformation. Each session follows the five-C structure: Connection, Content, Challenge, Change, Closure.

Audience

Participants include translators, translation consultants, language technology staff, exegetes, project facilitators, literacy workers, and organisational leadership. The three-lane content design reflects this range: content is differentiated for Translators, Consultants & Reviewers, and Technology & Leadership.

How to use the site

  • Course — five interactive session pages. Use these during a workshop session or for self-study.
  • Presentation mode — click "Presentation" in the nav (or press F11 for full-screen) to use any course page as a projector display. Arrow keys advance sections.
  • Handbook — the full AI Tools Handbook for deeper reading and reference.

Source & licensing

All source files (Markdown, HTML, CSS, JS) are publicly available in the GitHub repository. Content is available for use in Bible translation training contexts.
